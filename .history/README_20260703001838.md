# Gerenciador de Horários Acadêmicos

Sistema em Node.js, Express, EJS, Sequelize e PostgreSQL para gerenciamento de horários do curso de Engenharia de Computação.

## Funcionalidades

- Login por perfil: administrador e aluno.
- Dashboard administrativo com métricas do banco.
- Cadastro, edição e exclusão de professores.
- Cadastro, edição e exclusão de estudantes.
- Tela única de Disciplinas/Turmas.
- Relatório de turmas com editar/excluir.
- Relatório de carga docente.
- Configurações básicas.
- Validações básicas no backend e HTML.

## Como rodar

```bash
cp .env.example .env
npm install
sudo -u postgres createdb horarios_db
node app.js
```

Em outro terminal, para popular dados:

```bash
sudo -u postgres psql -d horarios_db -f sql/seed.sql
```

Acesse:

```txt
http://localhost:3000
```
